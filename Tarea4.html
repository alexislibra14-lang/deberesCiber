Bucles en python

Los bucles en Python, principalmente for y while, permiten repetir bloques de código eficientemente. El bucle for se usa para iterar sobre secuencias (listas, rangos, cadenas), mientras que while repite código mientras una condición sea verdadera. Se controlan con break (salir), continue (saltar iteración) y pass (bucle vacío

seguridad_bucle.py 
# Validación de contraseña con bucle
password_correcta = "alex123"
intento = ""

print("--- Sistema de Seguridad ---")

multiplos_5.py 
# Muestra los números del 1 al 50 que sean múltiplos de 5.
# 1. Definimos el punto de partida (el primer número de nuestra lista)
numero = 1

# 2. El bucle se ejecutará mientras 'numero' sea menor o igual a 50
while numero <= 50:
    
    # 3. Verificamos si el número es múltiplo de 5 usando el operador módulo (%)
    # Si el residuo de dividir 'numero' entre 5 es exactamente 0, es múltiplo
    if numero % 5 == 0:
        
        # 4. Si la condición del 'if' es verdadera, mostramos el número en pantalla
        print(numero)
    
    # 5. IMPORTANTE: Incrementamos el número en 1 en cada vuelta del bucle
    # Esto permite que el bucle avance del 1 al 2, del 2 al 3, y así sucesivamente
    numero += 1

# --- BLOQUE 1: NÚMEROS DEL 1 AL 10 ---
print("Números del 1 al 10:")
# 1. Definimos el inicio en 1
i = 1

# 2. El bucle se repite mientras 'i' sea menor o igual a 10
while i <= 10: 
    # 3. Imprimimos el valor actual de 'i'
    print(i)
    # 4. Sumamos 1 a 'i' en cada vuelta para avanzar
    i += 1 


# --- BLOQUE 2: NÚMEROS PARES DEL 2 AL 20 ---
print("\nNúmeros pares del 2 al 20:")
# 1. Empezamos en 2 porque es el primer par
numero = 2

# 2. El bucle se detendrá cuando 'numero' pase de 20
while numero <= 20: 
    # 3. Mostramos el número par actual
    print(numero)
    # 4. IMPORTANTE: Sumamos 2 para saltar directamente al siguiente par
    numero += 2 
    # Nota: Se eliminó la palabra 'cleae' que causaba error.


# --- BLOQUE 3: NÚMEROS DEL 10 AL 1 (DESCENDENTE) ---
print("\nNúmeros del 10 al 1 en orden descendente:")
# 1. Definimos el punto de partida (el más alto)
numero_desc = 10

# 2. El bucle funciona mientras el número no sea menor a 1
while numero_desc >= 1:
    # 3. Imprimimos el valor actual
    print(numero_desc)
    
    # 4. Restamos 1 en cada vuelta para ir hacia atrás
    numero_desc -= 1
suma_acumulada.py 
#Pide un número y calcula la suma acumulada desde 1 hasta ese valor
# 1. Pedimos el número límite al usuario
limite = int(input("Introduce un número para calcular la suma acumulada: "))

# 2. Inicializamos las variables
suma = 0
contador = 1

# 3. El bucle suma cada número hasta llegar al límite
while contador <= limite:
    suma += contador  # Sumamos el valor actual del contador a la bolsa
    contador += 1     # Pasamos al siguiente número

# 4. Mostramos el resultado final
print(f"\nLa suma acumulada desde 1 hasta {limite} es: {suma}")

#Muestra la tabla de multiplicar del número que ingrese el usuario.
# 1. Definimos el número de la tabla
numero = int(input("Ingresa un numero: "))

# 2. Inicializamos un contador en 1
contador = 1

print(f"\nTabla del {numero}:")

# 3. El bucle se ejecuta mientras el contador sea menor o igual a 12
while contador <= 12:
    resultado = numero * contador
    print(f"{numero} x {contador} = {resultado}")
    
    # 4. ¡Importante! Sumamos 1 al contador en cada vuelta
    contador += 1

#Imprime un triángulo de asteriscos de 5 filas (* / ** / *** …).
# 1. Creamos una variable 'filas_totales' para guardar el límite (cuántos pisos tendrá el triángulo)
filas_totales = 5 

# 2. Creamos un contador llamado 'fila_actual' que empieza en 1 (nuestro punto de partida)
fila_actual = 1 

# 3. Iniciamos el bucle 'while'. Se leería: "Mientras el número de la fila actual no pase de 5..."
while fila_actual <= filas_totales: 

    # 4. Creamos la variable 'triangulo'. 
    # Multiplicamos el texto "*" por el valor de 'fila_actual'. 
    # Si fila_actual es 3, Python genera "***".
    triangulo = "*" * fila_actual 

    # 5. La función 'print' muestra en la consola el contenido de la variable 'triangulo'
    print(triangulo) 

    # 6. sumamos 1 a 'fila_actual' (fila_actual = fila_actual + 1).
    # Esto hace que en la siguiente vuelta el triángulo tenga un asterisco más.
    # También evita que el bucle sea infinito, ya que eventualmente llegará a 6 y se detendrá.
    fila_actual += 1
